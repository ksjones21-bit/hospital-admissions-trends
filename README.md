# Hospital Admissions Trends Analysis

This project analyzes hospital admissions data to identify trends over time and compare admission types.

## Tools
Python, Pandas, Matplotlib

## Key Questions
- How do hospital admissions change over time?
- Which admission type occurs most frequently?

## Key Insights
Admissions remain relatively stable across months, with no strong seasonal patterns. Admission volumes are fairly evenly distributed across elective, urgent, and emergency categories.
